air-monitor-plots

A visual walk-through of the five standard air quality plots

The air-monitor-plots package turns hourly air quality data into Highcharts chart configurations. Each function takes plain data arrays plus a locationName and timezone and returns a configuration object — it does not draw the chart itself. The calling page passes that object to Highcharts.chart(container, config). All timestamps are UTC Luxon DateTime objects, and missing values are represented as null.

This page generates a 336-hour (14-day) synthetic PM2.5 dataset for America/Los_Angeles, modeling a clean-air baseline, a daily (diurnal) cycle, and a multi-day wildfire-smoke event peaking around day 7. The companion air-monitor-algorithms package computes the NowCast, daily, and diurnal inputs; everything is loaded from CDN with no build step.

Time series

timeseriesPlotConfig({ datetime, pm25, nowcast, locationName, timezone }) produces a line chart of raw hourly PM2.5 (gray markers) with the NowCast overlaid (black line). The y-axis carries colored AQI threshold lines, and the x-axis is rendered in the site's local timezone.

Hourly NowCast barplot

hourlyBarplotConfig({ datetime, nowcast, locationName, timezone }) draws each hour's NowCast value as a column colored by its AQI category — green for good air through maroon for hazardous. The knocked-out hours on day 12 appear as gaps.

Daily average barplot

dailyBarplotConfig({ daily_datetime, daily_mean, locationName, timezone }) shows one AQI-colored column per local calendar day. The rise and fall of the smoke event is clear across the two weeks.

Daily range barplot

dailyRangeBarplotConfig({ daily_datetime, daily_min, daily_mean, daily_max, locationName, timezone }) adds the daily min–max spread as a gray range bar, with the AQI-colored daily mean marked on top.

Diurnal profile (average day)

diurnalPlotConfig({ datetime, nowcast, hour_average, latitude, longitude, locationName, timezone }) plots the hour-of-day cycle: a multi-day average ("7 Day Mean") behind yesterday's and today's curves. Shaded bands mark night, computed from the site's latitude and longitude.

Small-multiples variants

Every plot has a small_* companion (for example small_timeseriesPlotConfig) that strips the legend and axis labels for compact "small multiples" displays.
