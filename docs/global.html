pm25_AQILines(width)

Returns an array of objects with {color, width, value} properties. These can be used to add colored AQI lines to plots with a PM2.5 axis measured in micrograms per meter cubed.

Usage in a Highcharts configuration:

yaxis: {
  plotlines: pm25_AQILines(),
}
Parameters:
Name Type Description
width number

Line width in pixels.
